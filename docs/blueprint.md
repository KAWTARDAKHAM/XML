# **App Name**: FluentGantt XML

## Core Features:

- Hybrid Data Persistence: Task management system backed by SQLite with full XML/XSD export and validation functionality for academic project requirements.
- Dynamic Gantt Timeline: Visual timeline view inspired by the user reference image, showing task duration, overlap, and current progress using CSS Grid.
- AI Productivity Consultant: Generative AI tool that analyzes task lists and XML structures to suggest deadline optimizations and priority rankings.
- XSLT Transformation Previewer: Built-in interface to apply XSLT stylesheets to task XML data for immediate HTML rendering and reports.
- Performance Statistics Hub: Dashboard with two primary chart visualizations showing task completion rates and effort distribution.
- Notification System: User profile configuration to receive task reminders via email.
- Fluent Authentication Flow: Modern login and sign-up interface following the dark, glassmorphic aesthetic provided.

## Style Guidelines:

- Dark mode theme: Background in deep onyx purple (#0F0D15) to match the dark professional aesthetic.
- Primary color: Radiant Indigo (#845EF7) for high contrast in navigation and primary timelines.
- Accent colors: Electric Cyan (#3BC9DB) for statistics and Rose Quartz (#FF8787) for urgent tasks.
- Font pairing: 'Space Grotesk' for technical-feeling headlines and labels; 'Inter' for clear, highly legible body text.
- Outline-style thin-stroke icons that respond with glowing transitions when hovered, maintaining the 'Fluent' design ethos.
- Sidereal layout with a collapsed left-hand navigation and expanded canvas for the Gantt timeline.
- Smooth 250ms ease-in-out transitions for card movement and task transformations.
